# Property Chronicle Design Direction

The interface should not feel like a friendly SaaS dashboard, a gothic horror site, or a generic AI-generated dark theme.

## Visual References

Blend:
- Soviet Late Modernism
- Constructivist poster composition
- Fauvist non-natural color blocking
- Institutional archive / public record office
- Weathered concrete, peeling posters, rust, water stains

## Mood

Serious, formal, historically heavy, cold, and restrained.

The emotional target is not fear. It is lingering presence, official decay, forgotten rituals, and temporal dislocation.

Avoid:
- Gothic cliches
- Cobwebs
- Ghost icons
- Halloween palettes
- Soft friendly SaaS cards
- Decorative UI that looks clickable but is not

## Shape Language

- Heavy geometric blocks
- Asymmetrical grids
- Constructivist diagonals
- Hard-edged panels
- Minimal rounding
- Monumental type
- Large empty pressure, not many small decorative parts

## Color

Use a desaturated, tense palette:
- tar black
- cold concrete gray
- washed ochre
- muted blood red
- deep earthy brown
- sudden off-white

Red should be structural, not decorative. It can frame, interrupt, or warn.

## Texture

Use subtle grain, concrete, worn poster, and water-stain overlays. Keep them low enough that data remains readable.

## Product Constraint

Admin and map screens must remain functional. The composition can feel severe, but workflows must be quiet and legible.
